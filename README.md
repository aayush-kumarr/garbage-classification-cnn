# Garbage Classification using CNN 
This project demonstrates a basics Convolutional Neural Network (CNN) built using TensorFlow and Keras to classify garbage images into six categories:
cardboard, glass, metal, paper, plastics and trash.

# Tools & Techniques 
- Python
- TensorFlow/Keras
- CNN
- ImageDataGenerator

# Files
- `Garbage_Classification_CNN.ipynb`- complete project implementation.

# Notes
The trained model file(`.h5`) was generated locally during execution.
It is not uploaded due to GitHub file size limitations.

# Conclusion
This project shows the practical application of CNNs for image classification in a beginner-friendly manner.
